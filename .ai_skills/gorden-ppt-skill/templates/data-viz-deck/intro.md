# 数据可视化合辑 · 大量原生图表

## 一句话
深蓝 + 砖红 + 浅灰 + 白底的数据看板模板，**预置 41 套真实业务数据可视化页面**（甘特图、漏斗图、柱图、饼图、环形图、折线图、面积图、热力图等），其中 **20+ 个页面含 PPT 原生 chart 对象，可程序化更新数据**。

## 风格标签
- 整体风格：**数据 / 看板 / 商务 / 信息密集**
- 排版复杂度：**很高**（每页多个图表 + 文字 KPI + 时间轴）
- 视觉元素：横向 / 纵向时间轴、KPI 圆环 (%)、柱状对比、饼图、环形进度、折线趋势、漏斗转化、甘特条
- 适合人群：数据分析师、运营 / 销售 / 财务、年度总结、商业 BI

## 注意事项
- **重头戏是改数据，不只是改文字**。如果你的实际数据和模板示例数据差异很大，柱图/饼图高度会不匹配 → 通过 `chart_data` 同步更新（看 chart-editing.md）。
- 大量页面信息密度很高，每页有 10-30 个文字标签。**字数控制极严**。
- 标题前缀（"数据展示/" "甘特图/" "数据图表/" "工作汇报/" 等）是模板自带的版式标识，**实际用时建议去掉前缀**只保留具体主题。
- 部分页面含 SmartArt 形状（非原生 chart），那些形状改不了数据，只能改文字。详见每页 `data_charts` 字段。
