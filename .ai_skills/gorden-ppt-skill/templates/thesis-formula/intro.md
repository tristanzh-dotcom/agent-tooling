# 开题报告万能公式 · 4 段式学术模板

## 一句话
米色奶咖底 + 深蓝深咖文字的学术风模板，按"研究背景 / 研究意义 / 研究现状 / 研究方法"4 段经典开题公式组织，**每段含 8-9 个不同学科的填好示例页**，让"研究小白"按公式套即可。

## 风格标签
- 整体风格：**学术 / 严谨 / 暖米色 / 极简版面**
- 排版复杂度：**中等**（每页是一种关系图 / 流程图样板）
- 视觉元素：暖米黄底 + 深海蓝标题 + 鱼骨图 / 雷达图 / 漏斗 / 金字塔 / 柱状图 / 数据图标 / 词云 / 时间线
- 适合人群：本/硕开题，特别是教育学 / 学前教育 / 护理学 / 智能制造 / AI 等多专业学生

## 主题色
- 主色：`#2A3F75`（深海蓝）
- 强调：`#C09548`（暖咖金）
- 底色：`#F6F0DC`（暖米黄）
- 文字：`#2B2B2B`（深灰）

## 字体
- 中文：思源黑体（粗体标题）
- 英文：Arial（标题用 Bold）

## 适用场景（不要被原标题限制）
- 本科 / 硕士 / 博士开题报告
- 课题立项书答辩
- 学科竞赛展示
- 教学/培训 - 教学生写开题报告

具体到学科，模板示例覆盖（仅作版式参考，并非限制）：
- 幼儿园 / 学前教育（多页）
- 中小学教学
- 护理学 / 医学
- 工程 / 物联网 / AI / 边缘计算
- 智慧农业 / 智慧社区
- 数据科学

## 注意事项
- 这是"按公式填空"型模板：4 段经典开题结构，每段你**只需挑 1 张最匹配你研究形态的版式**即可，不需要把同一段的多张全用上。
- 推荐组合：封面 (1) + 介绍页 (2，可跳过) + 各段扉页和 1 个示例 = 9-12 张完整开题报告。
- 每个示例的"内容文字"都是真实学科的研究表述，建议**重写而非微改**，否则会出现风马牛不相及的内容。
- 4 段扉页 (3/13/23/33) 是固定的格式："01 研究背景 RESEARCH BACKGROUND" 等，建议**保持序号 + 英文 + 仅修改章节顺序**。
- 含有多种数据可视化样式（鱼骨/雷达/金字塔等），属于 SmartArt 形状，**改文字 OK，改数据形状需要重画**。
