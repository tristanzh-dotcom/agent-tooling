# 麦肯锡风 · 咨询/分析专业模板

## 一句话
酱红 + 深蓝灰的咨询风专业模板（**与"高级感大厂"同款配色系列**），但版式更聚焦"逻辑结构"（金字塔 / 漏斗 / 对比 / 并列 / 总分），多达 37 页含真实业务案例。

## 风格标签
- 整体风格：**麦肯锡咨询 / 逻辑结构 / 高密度信息**
- 排版复杂度：**高**（每页 8-25 个文字模块 + 强结构化图形）
- 视觉元素：金字塔分级 / 漏斗模型 / 并列结构 / 对比结构 / 总分关系 / PDCA / 思维导图
- 适合人群：咨询顾问、战略部、MBA 学生、年终汇报、高管述职

## 主题色
- 主色：`#A52524`（酱红）
- 辅色：`#2D3955`（深蓝灰）
- 强调：`#D14D4A`（亮红）
- 底色：`#FFFFFF`（白）

## 字体
- 中文：思源黑体（加粗）
- 英文：Arial Black

## 适用场景（不要被原标题限制）
和 premium-corp 同源（同一系列的两份组合），区别：
- premium-corp: 偏"高级感互联网模板"，含各类业务案例
- mckinsey-style: 偏"咨询逻辑结构"，含金字塔/漏斗/并列/总分等典型麦肯锡式版式

具体场景：
- 咨询项目交付（管理咨询 / 战略咨询）
- 业务策略提案、高管述职
- 商业分析报告、行业研究
- 投资人路演（数据分析部分）
- MBA 商业案例分析
- 任何需要"金字塔结构 / 总分关系" 等逻辑可视化的演示

## 注意事项
- 比 premium-corp 多了"金字塔 / 漏斗 / 并列结构"等经典咨询版式，更适合做"结论+论据"型逻辑展示。
- 信息密度极高，**字数必须严格控制**，超过 max_chars 会破坏整张图。
- 每页"X 段内容页-XXX"标题前缀是版式说明，**使用时去掉前缀**只保留主题。
- 大多数图形是 SmartArt，**改文字 OK，改形状 / 颜色 / 节点数不行**。
- 金字塔图 (slide 37-40) 和漏斗图 (slide 31-33) 的层数 / 节点数是固定的；超过原数量请换版式。
