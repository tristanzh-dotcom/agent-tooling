# 汇报合辑·数据图表与业绩

## 一句话
深蓝商务的 38 页**数据图表 + 业绩汇报**子集。覆盖齿轮图、漏斗图、树状图、时间轴、市场活动转化、招聘漏斗、市场推广、销售业绩、第三季度总结、财务营收/成本、店铺流量、商品转化、关键指标完成情况等。

## 风格标签
- 整体风格：**蓝色商务 / 信息密集 / 全场景**（与 report-massive 系列其他两套同源，配色保持一致）
- 排版复杂度：**很高**（每页 10-30+ 文字块）
- 视觉元素：深蓝矩形 + 浅蓝填充 + 漏斗图 / 树状图 / 齿轮图 / 财务/营销/销售 / 转化分析
- 适合人群：所有职场汇报需求

## 主题色
- 主色：`#1E3A5F`（深海蓝）
- 强调：`#2F6EBA`（亮蓝）
- 辅色：`#A6B5C8`（浅蓝灰）
- 底色：`#FFFFFF`（白）

## 字体
- 中文：思源黑体 / 微软雅黑
- 英文/数字：Arial Bold

## 适用场景（不要被原标题限制）
漏斗图 / 树状图 / 齿轮图 / 财务/营销/销售 / 转化分析

本模板**全部 38 页可用**。原 v1.0.0 的封面/介绍/缩略图墙已在 **v1.0.2** 移除。

## 关联模板
本模板和 `report-massive-models` / `report-massive-charts` / `report-massive-reports` 系出同源（原 118 页超大合辑拆分），配色和字体保持一致。需要某类版式时优先在本套内挑，跨套混搭也无视觉冲突。
