# 运营 PPT 合辑 · 蓝色互联网运营

## 一句话
深蓝 + 浅蓝的运营汇报合辑，52 页**纯运营场景模板**（私域 / 用户 / 产品生命周期 / 营销活动 / 数据看板 / 项目管理 / 全域增长），适合互联网运营经理 / 增长黑客 / 电商团队。

## 风格标签
- 整体风格：**互联网运营 / 数据驱动 / 蓝色商务**
- 排版复杂度：**高**（每页 10-30 个文字模块 + 数据图）
- 视觉元素：深蓝矩形 + 数字 KPI + 柱图饼图折线 + 流程箭头 + 阶段时间线 + S 曲线（产品生命周期）
- 适合人群：互联网产品 / 运营 / 增长 / 电商团队

## 主题色
- 主色：`#1F3A93`（深蓝）
- 强调：`#3498DB`（亮蓝）
- 中性：`#ECF0F1`（浅灰）
- 文字：`#2C3E50`（深蓝灰）

## 字体
- 中文：思源黑体 / 微软雅黑
- 英文/数字：Arial Bold

## 适用场景（不要被原标题限制）
52 页覆盖运营的方方面面：
- **数据看板**：工作成果展示（多页，KPI + 趋势图 + 饼图）
- **运营策略**：私域运营策略模型、营销角色多样化、品牌全域运营
- **产品类**：产品生命周期、电商品牌扩展、产品销量
- **项目类**：项目管理、营销活动主题、战略规划、人员能力
- **数据汇报**：营收来源、销量增长、降本增效
- **季度 / 年度汇报**：2024 年工作开展、2024 年第二阶段目标
- **业绩**：销量增长、一月完成率、增量分析

具体场景：
- 月 / 季 / 年度运营汇报
- 产品复盘、活动复盘
- 电商业绩汇报、私域增长汇报
- 数据驱动决策汇报
- 增长团队 OKR / KPI 汇报
- 投资人月度更新（运营部分）

## 注意事项
- 大量页面共用"工作成果展示"标题，这是模板的通用版式名，**实际使用时需要换成具体页面主题**。
- 信息密度极高，文字字数控制严格。
- 该模板与 report-savior、data-viz-deck 有部分页面重叠/相似（同一作者风格），可互补选用。
- 含部分原生 chart（饼图、折线、柱图），可程序化更新数据。
