# 架构图合辑 · 领导超爱的方型/逻辑图

## 一句话
深蓝主调 + 浅蓝/白底的方形 / 逻辑图 / 框架图专项合辑，**预置 40 套大厂真实业务架构图**（项目管理 / 电商 / AI / 物流 / 供应链 / 营销 / DTC 等），适合给老板汇报"全景"、做体系化框架展示。

## 风格标签
- 整体风格：**架构图密集 / 商务正式 / 模块化 / 信息密度极高**
- 排版复杂度：**很高**（每页都是一张完整框架图，10-40 个文字模块）
- 视觉元素：深蓝色矩形 + 圆角 + 浅蓝色填充模块 + 箭头连接线 + 分组分层结构
- 适合人群：项目经理、产品经理、运营经理、咨询顾问、企业战略部

## 主题色
- 主色：`#1F3A93`（深海蓝）

- 浅色：`#E1ECF7`（浅蓝填充）

## 字体

## 注意事项
- 每页都是**信息密集**的复杂架构图，每个文字模块都很短（2-6 字标签为主）。**字数控制极其重要**，超出会把整张图毁掉。
- 大量页面是**框架嵌套+连接线**结构，**保持原图形布局不动**，只改文字。
- 选页时建议优先选"主题最接近"的版式，避免硬把通用框架套到完全不相关的业务领域。
- 如果你只是要"做一张架构图"而不是一份完整 PPT，**单页选用**就足够。
- 颜色变化方面，模板原色已经统一为蓝系，不建议自己改色（违和）。
