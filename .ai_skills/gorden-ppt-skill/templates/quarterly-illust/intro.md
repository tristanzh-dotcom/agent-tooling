# 蓝灰酸性插画 · 季度工作总结

## 一句话
黑白底色 + 亮蓝点缀 + 黑白手绘人物插画 + 像素 / Y2K / 复古浏览器窗口元素，"酸性设计 (acid graphic)" 风格的商务汇报模板，适合互联网公司 / 设计团队 / 创意行业的季度总结。

## 风格标签
- 整体风格：**酸性 / Y2K / 复古科技 / 设计感强 / 互联网风**
- 排版复杂度：**中等**（一页 3-4 个信息块，配大插画或浏览器窗口装饰）
- 视觉元素：黑白线稿插画人物（招手女生）、复古窗口框（按钮 + 滚动条 + 棋盘格）、亮蓝色 (#4F4FFF) 矩形高亮、几何斑驳网格、像素方格装饰
- 适合人群：互联网 / 科技 / 设计 / 数字营销 / 年轻一代汇报

## 主题色
- 主色：`#4F4FFF`（亮蓝）
- 黑色：`#000000`（线稿 + 标题）
- 白色：`#FFFFFF`（背景）
- 灰：`#D9D9D9`（中间灰）

## 字体
- 中文：思源黑体 / 微软雅黑 （加粗）
- 英文：Arial 黑体或字重重的 sans-serif

## 适用场景（不要被"季度工作总结"限制）
本模板视觉语言鲜明、有 Y2K / 设计感，特别适合：
- 互联网公司季度 / 年度 / 月度运营汇报
- 产品迭代复盘、需求评审
- 设计部门作品集 / 项目展示
- 数字营销活动复盘
- 创业项目 BP 中前几页
- 设计 / 内容 / 媒体团队的对外提案
- 年轻品牌的合作介绍
- 创意工作坊 / 设计周分享

## 模板包含的页面结构（共 19 页，全部可用）
- 1 个封面（slide 1：左标题 + 右插画女生 + LOGO/日期 + 汇报人按钮）
- 1 个目录（slide 2：左 "目录 CONTENTS" + 右 4 章节横向条形）
- 4 个章节扉页（slide 3 / 7 / 11 / 15）：左大插画 + 右大字章节标题
- 12 个内容版式（每章 3 页，slide 4-6 / 8-10 / 12-14 / 16-18）：
  - 多列卡片（2/3/4 列）+ 复古窗口装饰
  - 大段文字 + 小插画
  - 编号列表 (01/02/03) + 标题 + 多行说明
  - 复古浏览器窗口框 + 内嵌内容
- 1 个结束页（slide 19：感谢观看 THANK YOU + 插画女生）

## 不要选用的页面
本模板**全部 19 页都可用**，没有模板宣传页。

## 注意事项
- 章节标题最长 7-8 字效果最好，超过会挤压插画。
- 模板默认 4 章节："本季度工作概述 / 重点工作介绍 / 工作中不足之处 / 后期工作规划"——这是季度汇报的经典结构，也适合月度 / 项目复盘等场景。
- 黑白插画与亮蓝色块都是矢量形状，**位置和颜色保持不变**才能维持酸性设计感。
- 封面 / 结束页右上角的 "20XX/01/01" 是日期占位符，需要替换成真实日期。
- "YOUR LOGO" 顶部占位是 SmartArt 文字而非真实 logo 图，建议保留文字或替换为团队/公司名。
