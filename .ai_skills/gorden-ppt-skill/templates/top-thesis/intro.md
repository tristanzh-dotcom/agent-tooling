# 名校开题报告合辑 · 酒红学术风

## 一句话
酒红 + 米色调的学术风开题模板，按"研究背景 / 意义 / 现状 / 方法"4 段公式组织，**每段含 8-9 个真实学科示例**，比 thesis-formula 视觉更浓烈、配色更暖。

## 风格标签
- 整体风格：**学术 / 暖酒红 / 沉稳 / 高端**
- 排版复杂度：**中等偏高**（关系图 / 树形 / 旗帜信息图 / 数据图等多种形式）
- 视觉元素：酒红圆角矩形 + 米色背景 + 关系流程图 + 旗帜信息图 + 韦恩图 + 路径图

## 主题色
- 主色：`#7A2B22`（酒红）
- 强调：`#A55248`（中红）
- 底色：`#FAF1E4`（暖米色）
- 文字：`#2B2B2B`（深灰）

## 字体
- 中文：思源黑体 / 微软雅黑（加粗）
- 英文：Arial Bold

## 适用场景（不要被原标题限制）
和 thesis-formula 几乎完全相同（4 段公式开题），但**酒红色更适合**：
- 高校学生开题（视觉更"名校感"）
- 文科 / 教育学 / 心理学专业（色调温暖学术）
- 严肃学术答辩（不显得太活泼）

模板示例覆盖学科：教育学（小中高各学段）、护理学、医学、智能技术（AI / 机器人 / 视觉算法 / 物联网）等。

## 注意事项
- 与 thesis-formula 等价的"4 段开题公式"，只是视觉风格不同。两套模板的选择主要看色彩偏好。
- 每段建议挑 1 张最匹配你研究形态的版式即可。
- 4 段扉页 (3/12/21/31) 是固定格式，保持 "研究背景" 等中文标题。
