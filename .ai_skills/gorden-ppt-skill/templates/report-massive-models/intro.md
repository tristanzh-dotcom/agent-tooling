# 汇报合辑·思维模型与复盘

## 一句话
深蓝商务的 38 页**思维模型 + 个人复盘**子集。覆盖工作成果展示、SWOT、PDCA、KISS、问题分析与解决方案、不足与改进、复盘反思、项目流程、甘特图、鱼骨图等。最适合做季度/年度的**个人 / 项目复盘**。

## 风格标签
- 整体风格：**蓝色商务 / 信息密集 / 全场景**（与 report-massive 系列其他两套同源，配色保持一致）
- 排版复杂度：**很高**（每页 10-30+ 文字块）
- 视觉元素：深蓝矩形 + 浅蓝填充 + 思维模型 / SWOT/PDCA/KISS / 复盘 / 项目流程 / 鱼骨图 / 甘特图
- 适合人群：所有职场汇报需求

## 主题色
- 主色：`#1E3A5F`（深海蓝）
- 强调：`#2F6EBA`（亮蓝）
- 辅色：`#A6B5C8`（浅蓝灰）
- 底色：`#FFFFFF`（白）

## 字体
- 中文：思源黑体 / 微软雅黑
- 英文/数字：Arial Bold

## 适用场景（不要被原标题限制）
思维模型 / SWOT/PDCA/KISS / 复盘 / 项目流程 / 鱼骨图 / 甘特图
