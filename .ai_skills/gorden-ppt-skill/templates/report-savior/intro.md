# 汇报救命合辑 · 深蓝商务全场景

## 一句话
深蓝主调（融合 mckinsey-style + data-viz-deck 风格）的"全场景汇报救命包"，44 页覆盖**经典思维模型（SWOT/PEST/鱼骨/树状/房型）+ 业绩对比 + 项目进展 + 数据看板 + 述职**，是汇报场景的"瑞士军刀"。

## 风格标签
- 整体风格：**商务通用 / 深蓝 / 综合性 / 思维模型丰富**
- 排版复杂度：**中-高**（每页 5-20 个模块）
- 视觉元素：深蓝矩形 + 红色强调 + 数据看板 + SWOT 四象限 / 鱼骨图 / 房型图 / 树状图 / 甘特图
- 适合人群：综合管理者、咨询顾问、运营 / 销售 / 财务、年终述职者

## 主题色
- 主色：`#1F3A93`（深蓝）
- 强调：`#C0392B`（亮红）
- 辅色：`#34495E`（深蓝灰）
- 底色：`#FFFFFF`（白）

## 字体
- 中文：思源黑体 / 微软雅黑
- 英文/数字：Arial

## 适用场景（不要被原标题限制）
44 页几乎能覆盖所有商业汇报场景：
- **思维模型类**：SWOT 分析（slide 19）、PEST 分析（slide 20）、鱼骨图绩效目标（slide 22）、树状图（slide 13）、房型图（slide 15）
- **业绩 / 总结类**：成果扎实、全年业绩、季度总结、双驱突破等
- **项目类**：项目进展、阶段成果、五大举措、Q3 季度汇报
- **数据看板类**：人事 KPI、销售动态、生产成本、人员离职、预算使用、流量留量
- **述职类**：述职报告：不足之处与改进策略（slide 32, 35, 41）
- **甘特图**：年度计划推进总览（slide 24）

具体场景：
- 部门 / 项目 / 季度 / 年度汇报
- 述职竞聘、晋升答辩
- SWOT / PEST 战略分析
- 投资人月度汇报
- 业务复盘大会
- 部门述职、个人能力分析

## 注意事项
- 这是综合性"大全"模板，**适合按需挑页**：你的汇报场景缺什么版式就来这里挑。
- 选页时注意"风格一致"：避免在同一份 PPT 里把"卡通"和"严肃数据"混搭。本模板整体偏严肃商务。
- 大量使用 SmartArt 形状（思维模型 / 流程图 / 矩阵），改文字 OK，改形状 / 节点不行。
- 部分页面文字密度极高，**字数控制要严格**。
- 大量页面含数据图表占位，可使用 `chart_data` 程序化更新（详见 chart-editing.md，需要先检测每页 chart 对象）。
