# 高级感大厂 · PPT 模板合辑

## 一句话
酱红 + 深蓝灰 + 白底的高端商务配色，**预置 35+ 套战略分析、思维模型、产品运营框架**（PDCA / 冰山 / 金字塔 / 漏斗 / 双循环 / 甘特图 / 数据矩阵等），适合互联网 / 投行 / 咨询 / 战略部的高级感汇报。

## 风格标签
- 整体风格：**高级感 / 战略咨询 / 大厂商务 / 信息密集**
- 排版复杂度：**较高**（每页 5-15 个信息块 + 复杂图形）
- 视觉元素：酱红主形（#A52524 系）+ 深蓝灰辅形 + 数字图标 + 各类商业图（金字塔 / 漏斗 / 矩阵 / 循环 / 甘特）
- 适合人群：互联网中高层、战略分析师、咨询顾问、投行从业者、MBA 学生

## 主题色
- 主色：`#A52524`（酱红 / 中国红）
- 辅色：`#2D3955`（深蓝灰）
- 强调：`#D14D4A`（亮红）
- 底色：`#FFFFFF`（白）/ `#F5F5F5`（浅灰）

## 字体
- 中文：思源黑体 / 微软雅黑（标题加粗）
- 英文：Arial Black

## 适用场景（不要被原标题限制）
本合辑含 35 个**已经填充了真实商业案例**的版式，覆盖：
- **战略类**：上半年总结 / 下半年计划、年中数据分析、2025 战略分析、产品策划
- **运营类**：互联网产品运营循环图、KOC 种草、品牌策划推广、销售漏斗
- **管理类**：PDCA 循环、五段循环体系、工作流程汇报、甘特图
- **数据类**：数据图表、矩阵分析、冰山模型、年度数据图、用户画像
- **行业案例**：智能物流 / 新能源汽车 / 固态电池 / 智能供应链

具体场景：
- 季度 / 半年 / 年度战略汇报（投资人 / 董事会）
- 项目立项 / 产品评审 / 业务复盘
- 咨询提案、行业分析报告
- MBA 案例 / 商业分析作业
- 大厂晋升答辩、述职报告
- 任何需要"看起来很专业"的商业演示

## 模板包含的页面结构（共 35 页，全部可用）
全部 35 页都是真实商业案例样板，**没有封面 / 没有目录 / 没有结束页 / 没有模板宣传页**。按图形分类：
- **两段内容页**：A vs B 对比
- **三段内容页**：3 阶段递进
- **四段内容页**：4 步骤 / 4 象限
- **五段内容页**：5 层金字塔 / 五维分析
- **多段内容页**：6+ 模块复合图
- **专项页**：行业案例 / 甘特图 / 战略聚焦

每页 `detail.json` 都标了具体版式，使用时**按需挑页**，配合自己的 cover / agenda / ending 页（可从其他模板借用）组成完整 deck。

## 不要选用的页面
本模板**全部 35 页可用**。原 v1.0.0 的前 3 页（稻壳儿宣传 + 35 页精选广告 + 缩略图墙）已在 **v1.0.1** 移除。

## 注意事项
- 每页都是真实商业案例的样板，使用时**保留版式 + 替换文字**。具体案例的行业背景（如智能物流、固态电池）不影响版式套用——任何行业都可借鉴。
- 信息密度高，每页 max_chars 较紧；过长的文本会破坏整体高级感，建议**字字斟酌**。
- 大量图形（金字塔 / 漏斗 / 循环）是 SmartArt 形状组合，**改文字 OK，改形状 / 颜色 / 增减节点都不支持**。需要不同结构请换其他版式。
- 顶部 "X段内容页-XXX" 标题是版式描述，使用时**改成你的页面真实标题**（去掉 "X段内容页-" 前缀）。
- 主色酱红 + 深蓝灰是统一的高级配色，不要改色。
