# Capability Boundaries

Do not promise arbitrary beautification or one-click redesign for every PPT.

Supported:
- prompt-to-PPT via Gorden templates
- template-preserving text edits
- template replacement through extraction and remapping

Limited:
- decorative shape charts
- screenshot charts
- unsupported native chart types
- visual QA without LibreOffice and Poppler

Gorden built-in templates are non-commercial. Personal learning/research use is acceptable; commercial use requires separate template rights.
